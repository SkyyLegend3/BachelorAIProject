# Android Whisper Small Model (Bundle)

Lege hier optional das Whisper-Small-Modell fuer Android ab:

- Dateiname: `ggml-small-q5_1.bin`
- Zielpfad: `composeApp/src/androidMain/assets/models/ggml-small-q5_1.bin`

Wenn die Datei im Asset-Bundle liegt, kann die App das Modell lokal aus dem APK installieren.
Dann ist kein externer Download-Link erforderlich.

Hinweis:
- Die Datei ist gross. Fuer Team-Workflows empfiehlt sich Git LFS oder ein internes Artefakt-Hosting.
